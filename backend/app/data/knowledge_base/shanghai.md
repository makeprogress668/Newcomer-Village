# 上海旅行实用知识

## 外滩
- 最佳观景：晚 7:00-10:00 灯光秀
- 推荐位置：陈毅广场 / 外白渡桥
- 周末/节假日人极多，建议工作日傍晚

## 迪士尼
- 提前 60 天 APP 抢门票
- 旺季单日票 ¥499-799
- 必玩：创极速光轮 / 加勒比海盗 / 飞越地平线
- 早鸟票：开园前到场，先冲最热门

## 季节
- 春 (3-5月): 多雨，建议带伞
- 夏 (6-8月): 高温高湿 + 台风季
- 秋 (9-11月): 最佳季节
- 冬 (12-2月): 湿冷，需保暖防潮

## 交通
- 地铁 14 条线最完善，扫码乘车
- 磁悬浮：浦东机场 ¥50 / 8 分钟
- 出租车起步 ¥16，夜间 ¥21

## 美食
- 本帮菜：南翔小笼 / 老正兴 / 上海老饭店
- 海派西餐：红房子 / 德大
- 网红：外婆家 / 鹿港小镇
