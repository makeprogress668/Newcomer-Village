# 通用旅行知识

## 安全提示
- 重要证件 (身份证/护照) 复印件分开放
- 购买旅游保险 (建议平安/华泰)
- 紧急电话：报警 110 / 急救 120 / 火警 119 / 旅游投诉 12301

## 预算参考 (人均 / 天)
- 经济型: ¥300-500 (青旅 + 公共交通 + 小吃)
- 舒适型: ¥600-1000 (连锁酒店 + 网约车 + 中餐)
- 豪华型: ¥1500+ (五星 + 包车 + 米其林)

## 摄影建议
- 黄金时段: 日出后 1 小时 / 日落前 1 小时
- 蓝调时段: 日落后 30 分钟,城市夜景最美
- 备份: 云存储 + 内存卡双保险

## 防坑指南
- 黑车: 上车前问清价格,优先用滴滴/高德打车
- 黑导游: 景区门口陌生人主动搭讪要警惕
- 假门票: 只在官方公众号/官网/官方代销购票
- 宰客餐厅: 看大众点评 4.5 分以上 + 100+ 评论

## 退税 (境外游客)
- 单店满 500 元可申请,机场离境时办理
- 保留发票原件 + 商品 90 天内未拆封

## 小贴士
- 充电宝必须随身,不能托运 (容量 ≤100Wh)
- 高铁站务必提前 30 分钟到 (实名安检)
- 景区饮用水 ¥3-10,自带水壶可省钱
