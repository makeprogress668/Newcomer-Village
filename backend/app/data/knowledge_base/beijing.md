# 北京旅行实用知识

## 升旗仪式
- 时间：日出时分，夏季约 4:50-5:30，冬季约 7:00-7:30
- 地点：天安门广场
- 建议：提前 1 小时到达，过安检需 30 分钟
- 交通：地铁 1 号线天安门东/西站，凌晨地铁未运营时需打车
- 注意：禁止携带打火机/液体/无人机

## 故宫
- 提前 7-10 天官方公众号预约
- 周一闭馆（节假日除外）
- 推荐路线：午门 → 太和殿 → 中和殿 → 保和殿 → 御花园 → 神武门
- 游览时长：3-4 小时
- 旺季 4-10 月，淡季 11-3 月

## 长城
- 八达岭：交通最方便（S2 线/旅游专线），人最多
- 慕田峪：景观最美，缆车直达，外国游客多
- 司马台：保留原始风貌，适合摄影
- 不要同一天去两座长城（车程都需 1.5 小时）

## 季节穿衣
- 春 (3-5月): 风沙天多，建议口罩 + 长袖外套
- 夏 (6-8月): 高温多雨，防晒 + 雨具
- 秋 (9-11月): 最佳季节，温差大，加件外套
- 冬 (12-2月): 严寒干燥，羽绒服 + 保湿

## 交通
- 一卡通：可在地铁站机场购买，押金 ¥20
- 机场到市区：机场快线 ¥25 / 出租车 ¥120-150
- 高峰时段：早 7:30-9:30，晚 17:30-19:30 避开

## 美食推荐
- 烤鸭：全聚德 / 大董 / 便宜坊
- 涮羊肉：东来顺 / 聚宝源
- 老北京小吃：护国寺街 / 牛街
- 早餐：豆汁 / 焦圈 / 炒肝
